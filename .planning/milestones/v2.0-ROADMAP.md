## Milestone Summary

**Key Decisions:**
- Context-dependent GT is the core design change (weighted multi-signal scoring)
- "Bring Your Own System" is the primary workflow, built-in evaluators are examples
- Severity alone cannot determine GT (max score 0.25 < threshold 0.30)
- ~20% adversarial scenario injection to test context reasoning
- Shared description pools decouple text from GT class

**Issues Resolved:**
- Critical scenario leakage in v1.0 (single field predicted GT)
- Opaque aggregate score replaced with transparent metrics dashboard
- Output schema simplified (reasoning and processing_time now optional)
- Site-inappropriate categories eliminated (no shoplifting at solar farms)

**Issues Deferred:**
- Visual track v2 scenarios (v3.0)
- Multi-annotator ground truth (requires human annotators)
- Dispatch decisions and cost-aware scoring (v4.0)

**Technical Debt Resolved During Milestone:**
- Badge boundary mismatch (<=10 vs <10) — fixed
- Caltech v2 generation path — added
- Decision rubric field paths — corrected
- Missing CLI and adversarial tests — added
- Blocklist resample fallthrough — deterministic fallback added
