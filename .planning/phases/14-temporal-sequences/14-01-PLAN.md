CLASS STRUCTURE:

    class TemporalSequenceGenerator:
        def __init__(self, seed: int = 42):
            self.rng = np.random.RandomState(seed)
            self.seed = seed

        def generate(self, n_sequences: int = 50) -> list[dict]:
            alerts = []
            global_index = 0
            for seq_idx in range(n_sequences):
                pattern = ["monotonic_escalation", "escalation_then_resolution", "false_alarm"][seq_idx % 3]
                seq_alerts = self._build_sequence(seq_idx, pattern, global_index)
                alerts.extend(seq_alerts)
                global_index += len(seq_alerts)
            return alerts

INTERNAL HELPER _build_sequence(self, seq_idx, pattern, start_index):
  1. seq_length = self.rng.randint(3, 6)   # produces 3, 4, or 5
  2. turn_point = self.rng.randint(2, seq_length + 1)  # 1-indexed, range [2, seq_length]
     NOTE: randint(2, seq_length+1) gives values from 2 to seq_length inclusive.
     For seq_length=3, turn_point is 2 or 3. For seq_length=5, turn_point is 2-5.
     This varies the escalation/resolution position so no single position is always the peak.
  3. Sample SHARED per-sequence context (all alerts in sequence share these):
       - category: choose from UCF_CATEGORIES using self.rng
       - site_type: _sample_valid_site(category, self.rng)
       - base zone_sensitivity: choose from ["low", "medium", "high"] using self.rng
       - device: sample device_type and fpr consistent with existing generator patterns
       - weather: sample from WEATHER_POOL or equivalent
       - base_time_str: _generate_timestamp(base_time_of_day, self.rng) -- use "day" as base
  4. timestamp_base: parse base_time_str as datetime, then add (pos-1) * interval_minutes per alert.
       interval_minutes = self.rng.randint(5, 21)  -- sample ONCE per sequence, apply consistently.
       Use datetime.fromisoformat() to parse, add timedelta(minutes=interval_minutes*(pos-1)), format back with .isoformat().

SIGNAL TRAJECTORIES per pattern (pos is 1-indexed position within sequence):

monotonic_escalation:
  Before turn_point (pos < turn_point):
    severity = "LOW" or "MEDIUM" (alternate or rng choice)
    zone_type = "lobby" or "parking"
    time_of_day = "day" or "evening"
    badge_access_minutes_ago = None
    description from DESCRIPTION_POOL_AMBIGUOUS
  At and after turn_point (pos >= turn_point):
    severity = "HIGH"
    zone_type = "restricted"
    time_of_day = "night"
    badge_access_minutes_ago = None
    description from DESCRIPTION_POOL_UNAMBIGUOUS_THREAT

escalation_then_resolution:
  Before and at turn_point (pos <= turn_point):
    severity = "LOW" if pos == 1 else "HIGH"
    zone_type = "restricted"
    time_of_day = "night"
    badge_access_minutes_ago = None
    description from DESCRIPTION_POOL_UNAMBIGUOUS_THREAT if pos == turn_point else DESCRIPTION_POOL_AMBIGUOUS
  After turn_point (pos > turn_point):
    severity = "LOW"
    zone_type = "restricted"  (same zone, person was authorized)
    time_of_day = "night"     (same time)
    badge_access_minutes_ago = self.rng.randint(3, 10)   (fresh badge scan -- -0.45 signal)
    description from DESCRIPTION_POOL_UNAMBIGUOUS_BENIGN

false_alarm:
  First alert (pos == 1):
    severity = "HIGH" or "CRITICAL" (rng choice)
    zone_type = "restricted"
    time_of_day = "night"
    badge_access_minutes_ago = None
    description from DESCRIPTION_POOL_UNAMBIGUOUS_THREAT
  Subsequent alerts (pos > 1):
    severity = "LOW"
    zone_type = "parking" or "lobby" (rng choice)
    time_of_day = "day"
    badge_access_minutes_ago = self.rng.randint(1, 8)
    description from DESCRIPTION_POOL_UNAMBIGUOUS_BENIGN

For each position, call assign_ground_truth_v2() with the above signals.
Call _assign_difficulty() with the appropriate zone_sensitivity and time_of_day.
